## Overview
対象は「高齢者でゲームに馴染みのない方」です。</br>
以下の点を考慮してゲーム開発に取り組みました。

* 直感でわかりやすいUI
* 認知機能のトレーニング要素を考慮し、継続的に楽しんでもらえるゲーム

## Requirement
